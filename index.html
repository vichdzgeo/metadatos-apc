Guia para la creación de metadatos geográficos¶

GeoNetwork es una aplicación para la administración de recursos referenciados espacialmente, proporciona funciones de creación, edición y búsqueda de metadatos.

El catálogo de metadatos geográficos de APC se encuentra disponible en la siguiente dirección:

http://magrat.mine.nu:8088/geonetwork/
